    • Sobre o Site

    O projeto de desenvolvimento deste site foi iniciado para o aprendizado sobre as novas tecnologias HTML5 e CSS3.

    Ao estabelecer a hierarquia de navegação e a arquitetura de informação para um projeto web com 5 páginas, Aqui estão algumas etapas para essa definição: Menu de Navegação Principal, Priorização do Conteúdo, Criação de arquivos separados para cada página.

    Por fim utilizei fontes conhecidas da internet para conduzir a criação de site, Flaticon para icones, stack overflow para dúvidas, https://developer.mozilla.org - https://www.w3schools.com para pesquisa e também videos do youtube relacionados ao assunto.
